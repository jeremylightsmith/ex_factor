# ExactoKnife

Refactoring tools for elixir!

## Refactorings

### Sort aliases

```
mix refactor sort_aliases [FILE]
```

### Expand aliases

```
mix refactor expand_aliases [FILE]
```

### Consolidate aliases

```
mix refactor consolidate_aliases [FILE]
```


Many more to come...
